After copying all the modified code into the 'add' folder, future version updates can use the files under 'add' to add these features to the new version. The current features are as follows: 
1. Large Wooden Barrel 
Modeled after the large barrel feature in CarpettisAddition, large barrels are formed by placing barrels back-to-back (can be arranged horizontally or vertically).
PS: Hahaha, the full-item storage with large barrels that I've been longing to use can finally be used properly.

2. Quick-release action for dummy backpacks
A convenience feature for lazy people, no need to type commands every time.
PS: Honestly, it's just because I'm lazy.

3. Waxed copper box automatically stacks identical NBT shulker boxes
First, you need to enable shulker box stacking in leaves.

More features are in development, stay tuned.
